<model-viewer> - Physically Based Rendering Materials

This page includes several sample usages of the <model-viewer> element using higher-resolution models with multiple (normal, ambient occlusion, bump) maps with physically-based materials, and environment maps.

These examples use 16MB+ of assets. A local web server or landline connection is recommended.
